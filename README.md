# Pet Friend #

### 1. Описание ###
Приложение для мониторинга животных на передержке для последующего пристройства и помощи.

### 2. Экраны ###
  а. Splash Screen
  b. Registration / Login
  c. Add new announcement
  d. List of pets
  e. Settings
  
### Стек технологий ###
Kotlin, Dagger2, RxJava/ RxAndroid, DataBinding
